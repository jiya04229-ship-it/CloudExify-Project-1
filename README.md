CloudExify Cybersecurity Internship -  Project 1  Report

	Introduction 
     This project was completed as part of the CloudExify Cybersecurity Internship. The main objective was to create a virtual penetration testing environment and perform basic network security assessment.
The lab was created using Oracle VirtualBox with two virtual machines: Jiya (Kali Linux) as the attacker machine and Metasploitable 2 as the target machine. The project included IP identification, connectivity testing, Nmap scanning, and Wireshark traffic analysis.

	Objectives

o	Create a virtual cybersecurity lab using VirtualBox.
o	Configure Kali Linux and Metasploitable 2 machines.
o	Identify IP addresses and test connectivity.
o	Perform network scanning using Nmap.
o	Analyze network traffic using Wireshark.
o	Understand basic vulnerability identification.

	Tools Used

o	Oracle VirtualBox
o	Kali Linux
o	Metasploitable 2
o	Nmap
o	Wireshark

	Methodology
The project started by configuring both virtual machines in VirtualBox and connecting them through the same virtual network. The target machine's IP address was identified using the "ip a" command.
Ping testing was performed to verify communication between the machines. Different Nmap scans were conducted to discover open ports, running services, and service versions. Wireshark was used to capture and analyze network traffic during scanning activities.

	Key Findings
The scans successfully identified open ports and running services on Metasploitable 2. Nmap provided information about available services, while Wireshark helped analyze communication between the virtual machines.

	Skills Learned

o	Virtual lab setup
o	Linux command usage
o	IP address identification
o	Network scanning with Nmap
o	Packet analysis using Wireshark
o	Basic vulnerability assessment

	Conclusion
This project provided practical experience in penetration testing and network security assessment. It improved understanding of Linux tools, Nmap scanning, Wireshark analysis, and the basic methodology used in cybersecurity testing.
